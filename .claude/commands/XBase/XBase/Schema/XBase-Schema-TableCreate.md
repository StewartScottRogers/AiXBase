# XBase-Schema-TableCreate

Define a new table by adding its definition to `_schema.json` and creating an empty
`.ndjson` data file. No SQL DDL is generated or executed.

## Inputs

| Parameter | Type | Required | Default | Description |
|---|---|---|---|---|
| `ConnectionName` | string | yes | — | Open connection alias |
| `TableName` | string | yes | — | Name of the table to create (alphanumeric + underscore) |
| `Columns` | array | yes | — | Array of column definition objects (see below) |
| `IfNotExists` | bool | no | `true` | Succeed silently when the table already exists |

### Column Definition Object

| Field | Type | Required | Default | Description |
|---|---|---|---|---|
| `Name` | string | yes | — | Column name (alphanumeric + underscore) |
| `Type` | string | yes | — | `INTEGER`, `TEXT`, `REAL`, `BLOB`, or `NUMERIC` |
| `PrimaryKey` | bool | no | `false` | Mark as the primary key column |
| `AutoIncrement` | bool | no | `false` | Auto-assign via `NextId` counter in `_schema.json` |
| `Nullable` | bool | no | `true` | If `false`, enforce NOT NULL on insert |
| `Unique` | bool | no | `false` | Enforce uniqueness on insert and update |
| `Default` | any | no | — | Value to use when the column is omitted on insert |
| `ForeignKey` | string | no | — | `TableName.ColumnName` reference for FK validation |

## Outputs

```json
{
  "Success": true,
  "TableName": "<name>",
  "CreatedAt": "<ISO-8601>"
}
```

## Steps

1. Validate `ConnectionName`; if not registered, return `XBASE_CONNECTION_INVALID`
2. Validate `TableName` (alphanumeric + underscore, non-empty)
3. `File.ReadAllText(_schema.json)`; parse JSON
4. If a table entry with `Name == TableName` already exists:
   - If `IfNotExists` is `true`: return `Success: true` immediately
   - Otherwise: return `XBASE_SCHEMA_TABLE_EXISTS`
5. If no column has `PrimaryKey: true`, prepend an implicit column:
   `{ Name: "Id", Type: "INTEGER", PrimaryKey: true, AutoIncrement: true, Nullable: false, Unique: true, Default: null, ForeignKey: null }`
6. Append these columns if not already present: `CreatedAt TEXT`, `UpdatedAt TEXT`, `IsDeleted INTEGER DEFAULT 0`
7. Validate each column definition; return `XBASE_SCHEMA_COLUMN_INVALID` on any malformed entry
8. Add the table entry to the `Tables` array with `NextId: 1`
9. `File.WriteAllText(_schema.json, updatedSchema)`
10. `File.WriteAllText({TableName}.ndjson, "")` — create an empty data file
11. Return `TableName` and `CreatedAt`

## Script Templates

### PowerShell

```powershell
# Generated by XBase-Schema-TableCreate
param($DatabasePath, $TableName, $ColumnsJson)
$schema    = Get-Content "$DatabasePath\_schema.json" -Raw | ConvertFrom-Json
$columns   = $ColumnsJson | ConvertFrom-Json
$now       = (Get-Date).ToUniversalTime().ToString("yyyy-MM-ddTHH:mm:ssZ")
$tableDef  = @{ Name = $TableName; NextId = 1; Columns = $columns }
$schema.Tables += $tableDef
$schema | ConvertTo-Json -Depth 20 | Set-Content "$DatabasePath\_schema.json" -Encoding UTF8
"" | Set-Content "$DatabasePath\$TableName.ndjson" -Encoding UTF8
```

### Python

```python
# Generated by XBase-Schema-TableCreate
import json, pathlib, datetime

def create_table(db_path, table_name, columns):
    dbpath  = pathlib.Path(db_path)
    schema  = json.loads((dbpath / "_schema.json").read_text())
    now     = datetime.datetime.utcnow().strftime("%Y-%m-%dT%H:%M:%SZ")
    schema["Tables"].append({"Name": table_name, "NextId": 1, "Columns": columns})
    (dbpath / "_schema.json").write_text(json.dumps(schema, indent=2))
    (dbpath / f"{table_name}.ndjson").write_text("", encoding="utf-8")
    return now
```

## Error Codes

| Code | Condition |
|---|---|
| `XBASE_CONNECTION_INVALID` | `ConnectionName` not open |
| `XBASE_SCHEMA_TABLE_EXISTS` | Table already exists and `IfNotExists` is `false` |
| `XBASE_SCHEMA_COLUMN_INVALID` | Column definition is malformed or contains duplicate names |

## Dependencies

- `XBase-Database-Connect`
