# XBase-Database-Initialize

Create a new database directory with metadata and an empty schema.

## Inputs

| Parameter | Type | Required | Default | Description |
|---|---|---|---|---|
| `DatabaseName` | string | yes | — | Name of the directory to create under `XBaseFiles/` |
| `OverwriteIfExists` | bool | no | `false` | Delete and recreate if the directory already exists |

## Outputs

```json
{
  "Success": true,
  "DatabasePath": "<absolute path to the database directory>",
  "CreatedAt": "<ISO-8601>"
}
```

## Steps

1. Reference the `XBaseRuntimeContext` established by `XBase-Runtime-Detect`. If no runtime context is present in the session, run `XBase-Runtime-Detect` first.
2. Validate `DatabaseName`: reject if empty or if it contains `..`, `/`, `\`, `:`, `*`, `?`, `"`, `<`, `>`, `|`, or null bytes — return `XBASE_DATABASE_PATH_INVALID`.
3. Resolve the absolute path: `XBaseFiles/{DatabaseName}/` relative to the project root.
4. If the directory already exists and `OverwriteIfExists` is `false`, return `XBASE_DATABASE_EXISTS`.
5. If the directory already exists and `OverwriteIfExists` is `true`, delete it recursively using the appropriate script (see Script Templates below).
6. Create the database directory.
7. Write `_meta.json` with content: `{ "XBaseVersion": 1, "Name": "<DatabaseName>", "CreatedAt": "<now ISO-8601>", "UpdatedAt": "<now ISO-8601>" }`
8. Write `_schema.json` with content: `{ "Tables": [], "Indexes": [] }`
9. Return `DatabasePath` and `CreatedAt`.

## Script Templates

### PowerShell

```powershell
# Generated by XBase-Database-Initialize
$dbPath = Join-Path $PSScriptRoot "XBaseFiles" "<DatabaseName>"
if (Test-Path $dbPath) { Remove-Item $dbPath -Recurse -Force }  # only when OverwriteIfExists
New-Item -ItemType Directory -Path $dbPath -Force | Out-Null
$now = (Get-Date).ToUniversalTime().ToString("yyyy-MM-ddTHH:mm:ssZ")
$meta = @{ XBaseVersion = 1; Name = "<DatabaseName>"; CreatedAt = $now; UpdatedAt = $now }
Set-Content -Path (Join-Path $dbPath "_meta.json") -Value ($meta | ConvertTo-Json -Compress) -Encoding UTF8
Set-Content -Path (Join-Path $dbPath "_schema.json") -Value '{"Tables":[],"Indexes":[]}' -Encoding UTF8
Write-Output "{ `"Success`": true, `"DatabasePath`": `"$dbPath`", `"CreatedAt`": `"$now`" }"
```

### bash

```bash
#!/usr/bin/env bash
# Generated by XBase-Database-Initialize
DB_PATH="$(pwd)/XBaseFiles/<DatabaseName>"
rm -rf "$DB_PATH"   # only when OverwriteIfExists
mkdir -p "$DB_PATH"
NOW=$(date -u +"%Y-%m-%dT%H:%M:%SZ")
echo "{\"XBaseVersion\":1,\"Name\":\"<DatabaseName>\",\"CreatedAt\":\"$NOW\",\"UpdatedAt\":\"$NOW\"}" > "$DB_PATH/_meta.json"
echo '{"Tables":[],"Indexes":[]}' > "$DB_PATH/_schema.json"
echo "{\"Success\":true,\"DatabasePath\":\"$DB_PATH\",\"CreatedAt\":\"$NOW\"}"
```

### Python

```python
# Generated by XBase-Database-Initialize
import os, json, shutil
from datetime import datetime, timezone

db_path = os.path.join(os.getcwd(), "XBaseFiles", "<DatabaseName>")
if os.path.exists(db_path):  # only when OverwriteIfExists
    shutil.rmtree(db_path)
os.makedirs(db_path)
now = datetime.now(timezone.utc).strftime("%Y-%m-%dT%H:%M:%SZ")
with open(os.path.join(db_path, "_meta.json"), "w", encoding="utf-8") as f:
    json.dump({"XBaseVersion": 1, "Name": "<DatabaseName>", "CreatedAt": now, "UpdatedAt": now}, f, separators=(",",":"))
with open(os.path.join(db_path, "_schema.json"), "w", encoding="utf-8") as f:
    json.dump({"Tables": [], "Indexes": []}, f, separators=(",",":"))
print(json.dumps({"Success": True, "DatabasePath": db_path, "CreatedAt": now}))
```

## Error Codes

| Code | Condition |
|---|---|
| `XBASE_DATABASE_EXISTS` | Directory already exists and `OverwriteIfExists` is `false` |
| `XBASE_DATABASE_PATH_INVALID` | Name contains illegal characters or path traversal sequences |

## Dependencies

- `XBase-Runtime-Detect` — session must have a runtime context before this skill executes.
