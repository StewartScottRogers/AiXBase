# XBase-Database-Disconnect

Deregister a named connection alias from the session. Optionally rolls back any open
transaction workspaces before closing.

## Inputs

| Parameter | Type | Required | Default | Description |
|---|---|---|---|---|
| `ConnectionName` | string | yes | — | The logical alias registered by `XBase-Database-Connect` |
| `RollbackOpenTransactions` | bool | no | `true` | If `true`, delete any open `_txn_{name}/` workspace directories before deregistering |

## Outputs

```json
{
  "Success": true,
  "ConnectionName": "<alias>",
  "ClosedAt": "<ISO-8601>"
}
```

## Steps

1. Look up `ConnectionName` in the session; if not found, return `XBASE_CONNECTION_INVALID`
2. Resolve the `DatabasePath` mapped to this `ConnectionName`
3. Scan the database directory for any `_txn_*/` workspace directories:
   - If `RollbackOpenTransactions` is `true`: `Directory.Delete(_txn_*/,  recursive: true)` for each workspace found
   - If `RollbackOpenTransactions` is `false` and any `_txn_*/` directory exists: return `XBASE_TRANSACTION_STILL_OPEN`
4. Deregister `ConnectionName` from the session
5. Return `ClosedAt`

## Script Templates

### PowerShell

```powershell
# Generated by XBase-Database-Disconnect
param($DatabasePath, $RollbackOpenTransactions = $true)
$txns = Get-ChildItem -Path $DatabasePath -Directory -Filter "_txn_*" -ErrorAction SilentlyContinue
foreach ($txn in $txns) {
    if ($RollbackOpenTransactions) {
        Remove-Item -Path $txn.FullName -Recurse -Force
    } else {
        throw "XBASE_TRANSACTION_STILL_OPEN: $($txn.Name)"
    }
}
```

### Python

```python
# Generated by XBase-Database-Disconnect
import pathlib, shutil

def disconnect(db_path, rollback_open=True):
    db = pathlib.Path(db_path)
    for txn in db.glob("_txn_*"):
        if txn.is_dir():
            if rollback_open:
                shutil.rmtree(txn)
            else:
                raise RuntimeError(f"XBASE_TRANSACTION_STILL_OPEN: {txn.name}")
```

## Error Codes

| Code | Condition |
|---|---|
| `XBASE_CONNECTION_INVALID` | `ConnectionName` not found in the session |
| `XBASE_TRANSACTION_STILL_OPEN` | Open transaction workspace exists and `RollbackOpenTransactions` is `false` |

## Dependencies

- `XBase-Database-Connect` — must have been called to open this connection
