# XBase-Transaction-Savepoint

Create a named savepoint within an open transaction by copying the current workspace
state into a `sp_{SavepointName}/` subdirectory.

## Inputs

| Parameter | Type | Required | Default | Description |
|---|---|---|---|---|
| `TransactionName` | string | yes | — | Logical alias of the enclosing transaction |
| `SavepointName` | string | yes | — | Name for this savepoint (alphanumeric + underscore) |

## Outputs

```json
{
  "Success": true,
  "TransactionName": "<alias>",
  "SavepointName": "<name>",
  "CreatedAt": "<ISO-8601>"
}
```

## Steps

1. Validate `TransactionName` is active in the session; if not, return `XBASE_TRANSACTION_NOT_OPEN`
2. Resolve workspace path: `{DatabasePath}/_txn_{TransactionName}/`
3. Verify the workspace directory exists on disk; if not, return `XBASE_TRANSACTION_NOT_OPEN`
4. Resolve savepoint path: `{DatabasePath}/_txn_{TransactionName}/sp_{SavepointName}/`
5. If the savepoint directory already exists, return `XBASE_SAVEPOINT_NAME_IN_USE`
6. `Directory.CreateDirectory(sp_{SavepointName}/)`
7. Copy every **file** from the workspace root into `sp_{SavepointName}/` using `File.Copy(src, dest)` — do **not** recurse into existing `sp_*/` subdirectories
8. Return `SavepointName` and `CreatedAt`

## Script Templates

### PowerShell

```powershell
# Generated by XBase-Transaction-Savepoint
param($DatabasePath, $TransactionName, $SavepointName)
$workspace  = "$DatabasePath\_txn_$TransactionName"
$savepoint  = "$workspace\sp_$SavepointName"
New-Item -ItemType Directory -Path $savepoint | Out-Null
Get-ChildItem -Path $workspace -File | ForEach-Object {
    Copy-Item -Path $_.FullName -Destination "$savepoint\$($_.Name)"
}
```

### Python

```python
# Generated by XBase-Transaction-Savepoint
import pathlib, shutil

def create_savepoint(db_path, transaction_name, savepoint_name):
    workspace = pathlib.Path(db_path) / f"_txn_{transaction_name}"
    savepoint = workspace / f"sp_{savepoint_name}"
    savepoint.mkdir()
    for f in workspace.iterdir():
        if f.is_file():
            shutil.copy2(f, savepoint / f.name)
```

## Error Codes

| Code | Condition |
|---|---|
| `XBASE_TRANSACTION_NOT_OPEN` | No active transaction with that name |
| `XBASE_SAVEPOINT_NAME_IN_USE` | A savepoint with that name already exists in this transaction |
