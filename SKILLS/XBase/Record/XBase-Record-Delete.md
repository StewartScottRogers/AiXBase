# XBase-Record-Delete

Delete rows matching a filter. Performs a soft delete by default (sets `IsDeleted = 1`).
Pass `HardDelete: true` to remove matching lines from the `.dbf` file entirely.
A `Filter` is always required to prevent unintentional full-table deletes.

## Inputs

| Parameter | Type | Required | Default | Description |
|---|---|---|---|---|
| `ConnectionName` | string | yes | â€” | Open connection alias |
| `TableName` | string | yes | â€” | Target table |
| `Filter` | object | yes | â€” | Compiled filter from `XBase-Query-Filter` â€” required |
| `HardDelete` | bool | no | `false` | If `true`, remove matching lines from the file entirely |
| `TransactionName` | string | no | â€” | Execute within this named transaction |

## Outputs

```json
{
  "Success": true,
  "DeletedCount": 1,
  "HardDelete": false
}
```

## Steps

1. Validate `ConnectionName`; if not registered, return `XBASE_CONNECTION_INVALID`
2. Validate `Filter` is non-empty; return `XBASE_RECORD_FILTER_REQUIRED` if absent or empty
3. Resolve the active data directory (transaction workspace if `TransactionName` supplied; copy `.dbf` lazily if needed)
4. `File.ReadAllText(_schema.json)`; locate `TableName`; if absent, return `XBASE_SCHEMA_TABLE_NOT_FOUND`
5. `File.ReadAllBytes({TableName}.dbf)`; read DBF header to obtain `HeaderSize`, `RecordSize`, `RecordCount`, and field descriptors; decode each record from its fixed-width byte positions line as JSON
6. For each row, apply the `Filter` specification:
   - **Soft delete** (`HardDelete: false`): set `IsDeleted = 1` and `UpdatedAt` to current ISO-8601 timestamp; keep the line in the output
   - **Hard delete** (`HardDelete: true`): omit the matching line from the output entirely
   - Increment `DeletedCount` for each matched row
7. `File.WriteAllBytes({TableName}.dbf, packedBytes)`; write compacted DBF excluding deletion-flagged records; update header record count
8. Rebuild `.ndx` files for all indexes on this table when `HardDelete` is `true`; soft-delete leaves index entries intact since rows remain in the file
9. Return `DeletedCount` and `HardDelete` mode used

## Script Templates

### PowerShell

```powershell
# Generated by XBase-Record-Delete
param($DatabasePath, $TableName, $FieldName, $FieldValue, [bool]$HardDelete = $false)
$now   = (Get-Date).ToUniversalTime().ToString("yyyy-MM-ddTHH:mm:ssZ")
$path  = "$DatabasePath\$TableName.dbf"
$lines = Get-Content $path -Encoding UTF8 | Where-Object { $_ -ne "" }
$out = foreach ($line in $lines) {
    $row = $line | ConvertFrom-Json
    if ($row.$FieldName -eq $FieldValue) {
        if ($HardDelete) { continue }
        $row.IsDeleted = 1; $row.UpdatedAt = $now
        $row | ConvertTo-Json -Compress
    } else { $line }
}
$out | Set-Content $path -Encoding UTF8
```

### Python

```python
# Generated by XBase-Record-Delete
import json, datetime, pathlib

def delete_rows(db_path, table_name, filter_fn, hard_delete=False):
    dbpath = pathlib.Path(db_path)
    path   = dbpath / f"{table_name}.dbf"
    now    = datetime.datetime.utcnow().strftime("%Y-%m-%dT%H:%M:%SZ")
    rows   = [json.loads(l) for l in path.read_text().splitlines() if l.strip()]
    count, out = 0, []
    for row in rows:
        if filter_fn(row):
            count += 1
            if not hard_delete:
                row["IsDeleted"] = 1
                row["UpdatedAt"] = now
                out.append(row)
        else:
            out.append(row)
    path.write_text("\n".join(json.dumps(r, separators=(",",":")) for r in out) + "\n")
    return count
```

## Error Codes

| Code | Condition |
|---|---|
| `XBASE_CONNECTION_INVALID` | Connection not open |
| `XBASE_SCHEMA_TABLE_NOT_FOUND` | Table does not exist |
| `XBASE_RECORD_FILTER_REQUIRED` | `Filter` was empty or omitted |
| `XBASE_TRANSACTION_NOT_OPEN` | `TransactionName` supplied but workspace not found |
