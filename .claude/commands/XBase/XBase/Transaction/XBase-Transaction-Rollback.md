# XBase-Transaction-Rollback

Roll back an open transaction by deleting its workspace directory (full rollback) or
restoring from a savepoint snapshot (partial rollback).

## Inputs

| Parameter | Type | Required | Default | Description |
|---|---|---|---|---|
| `TransactionName` | string | yes | — | Logical alias registered by `XBase-Transaction-Begin` |
| `ToSavepoint` | string | no | — | If provided, restore to this savepoint and leave the transaction open |

## Outputs

```json
{
  "Success": true,
  "TransactionName": "<alias>",
  "RolledBackTo": "full",
  "RolledBackAt": "<ISO-8601>"
}
```

`RolledBackTo` is `"full"` or the savepoint name.

## Steps

**Full rollback** (`ToSavepoint` not supplied):

1. Look up `TransactionName` in the session; if not found, return `XBASE_TRANSACTION_NOT_OPEN`
2. Verify `{DatabasePath}/_txn_{TransactionName}/` exists on disk; if not, return `XBASE_TRANSACTION_NOT_OPEN`
3. `Directory.Delete({DatabasePath}/_txn_{TransactionName}/, recursive: true)` — discards all workspace files and savepoint subdirectories
4. Deregister `TransactionName` from the session
5. Return `RolledBackTo: "full"` and `RolledBackAt`

**Savepoint rollback** (`ToSavepoint` supplied):

1. Look up `TransactionName`; if not found, return `XBASE_TRANSACTION_NOT_OPEN`
2. Resolve savepoint path: `{DatabasePath}/_txn_{TransactionName}/sp_{ToSavepoint}/`
3. If the savepoint directory does not exist, return `XBASE_SAVEPOINT_NOT_FOUND`
4. For each file in `sp_{ToSavepoint}/`: `File.Copy(savepointFile, workspaceFile, overwrite: true)`
5. Delete savepoint directories that were created after `ToSavepoint` (preserve `sp_{ToSavepoint}/` itself)
6. Leave the transaction open — do **not** deregister `TransactionName`
7. Return `RolledBackTo: ToSavepoint` and `RolledBackAt`

## Script Templates

### PowerShell

```powershell
# Generated by XBase-Transaction-Rollback (full rollback)
param($DatabasePath, $TransactionName)
$workspace = "$DatabasePath\_txn_$TransactionName"
if (Test-Path $workspace) { Remove-Item -Path $workspace -Recurse -Force }
```

### Python

```python
# Generated by XBase-Transaction-Rollback (full rollback)
import pathlib, shutil

def rollback_transaction(db_path, transaction_name):
    workspace = pathlib.Path(db_path) / f"_txn_{transaction_name}"
    if workspace.is_dir():
        shutil.rmtree(workspace)
```

## Error Codes

| Code | Condition |
|---|---|
| `XBASE_TRANSACTION_NOT_OPEN` | No active transaction with that name |
| `XBASE_SAVEPOINT_NOT_FOUND` | `ToSavepoint` directory does not exist in the workspace |

## Dependencies
