# XBase-Record-Select

Retrieve rows from a table by reading its `.dbf` file and evaluating filter, sort,
join, aggregate, and pagination specifications entirely in memory against parsed JSON
objects. No database engine is required.

## Inputs

| Parameter | Type | Required | Default | Description |
|---|---|---|---|---|
| `ConnectionName` | string | yes | — | Open connection alias |
| `TableName` | string | yes | — | Source table |
| `Columns` | array | no | `["*"]` | Column names to return; `"*"` returns all columns |
| `Filter` | object | no | — | Compiled filter from `XBase-Query-Filter` |
| `Sort` | object | no | — | Compiled sort from `XBase-Query-Sort` |
| `Join` | object | no | — | Compiled join from `XBase-Query-Join` |
| `Aggregate` | object | no | — | Compiled aggregate from `XBase-Query-Aggregate` |
| `Limit` | int | no | — | Maximum rows to return after pagination |
| `Offset` | int | no | `0` | Rows to skip before returning results |
| `IncludeDeleted` | bool | no | `false` | If `false`, exclude rows where `IsDeleted = 1` |
| `TransactionName` | string | no | — | Read from this transaction's workspace if supplied |

## Outputs

```json
{
  "Success": true,
  "Rows": [ { "Id": 1, "Summary": "..." } ],
  "TotalCount": 57
}
```

`TotalCount` is the count of all matching rows before `Limit`/`Offset` are applied.

## Steps

1. Validate `ConnectionName`; if not registered, return `XBASE_CONNECTION_INVALID`
2. Resolve the active data directory (transaction workspace if `TransactionName` supplied, else live)
3. `File.ReadAllText(_schema.json)`; locate the `TableName` definition; if absent, return `XBASE_SCHEMA_TABLE_NOT_FOUND`
4. Validate each name in `Columns` against the table definition; return `XBASE_SCHEMA_COLUMN_MISSING` on unknown column (skip if `"*"` present)
5. `File.ReadAllBytes({TableName}.dbf)`; read DBF header to obtain `HeaderSize`, `RecordSize`, `RecordCount`, and field descriptors; decode each record from its fixed-width byte positions line as JSON
6. Exclude rows where `IsDeleted == 1` unless `IncludeDeleted` is `true`
7. Apply `Filter` specification in memory — for each row evaluate recursively:
   - `=` `!=` `<` `<=` `>` `>=`: compare `row[Field]` to `Value` using appropriate type coercion
   - `LIKE`: match `row[Field]` against `Value` pattern where `%` = any chars, `_` = one char; case-insensitive
   - `IN` / `NOT IN`: check whether `row[Field]` is in / not in the `Value` array
   - `IS NULL` / `IS NOT NULL`: check presence and nullness of `row[Field]`
   - Nested `Filters`: combine child results with `LogicalOperator` (`AND` / `OR`)
8. Apply `Join` specification in memory (if supplied):
   - Read `TargetTable.dbf`; parse each line; exclude soft-deleted rows
   - `INNER`: keep only primary rows with at least one match on `OnLeft == OnRight`; merge join-table fields into the row
   - `LEFT`: keep all primary rows; merge matched join-table fields where a match exists; null for unmatched fields
9. Apply `Aggregate` specification in memory (if supplied):
   - Group rows by `GroupBy` column values
   - Compute `COUNT`, `SUM`, `AVG`, `MIN`, `MAX` per group; return aggregated rows
10. Set `TotalCount` = length of the row set after steps 6–9
11. Apply `Sort`: sort by each column in `Sort.Columns` in specified `Direction` (`ASC` / `DESC`)
12. Apply `Offset`: skip the first `Offset` rows
13. Apply `Limit`: keep at most `Limit` rows (if supplied)
14. Project `Columns`: if not `["*"]`, return only the named fields from each row
15. Return `Rows` and `TotalCount`

## Script Templates

### PowerShell

```powershell
# Generated by XBase-Record-Select (core read + soft-delete filter)
param($DatabasePath, $TableName, $IncludeDeleted = $false)
$rows = Get-Content "$DatabasePath\$TableName.dbf" -Encoding UTF8 |
    Where-Object { $_ -ne "" } |
    ForEach-Object { $_ | ConvertFrom-Json }
if (-not $IncludeDeleted) {
    $rows = $rows | Where-Object { $_.IsDeleted -eq 0 }
}
$rows
```

### Python

```python
# Generated by XBase-Record-Select
import json, re, pathlib

def select_rows(db_path, table_name, include_deleted=False,
                filter_spec=None, sort_spec=None, limit=None, offset=0):
    dbf = pathlib.Path(db_path) / f"{table_name}.dbf"
    rows   = [json.loads(l) for l in dbf.read_text().splitlines() if l.strip()]
    if not include_deleted:
        rows = [r for r in rows if r.get("IsDeleted", 0) == 0]
    if filter_spec:
        rows = [r for r in rows if _match(r, filter_spec)]
    if sort_spec:
        for col in reversed(sort_spec.get("Columns", [])):
            rev = col.get("Direction", "ASC") == "DESC"
            rows.sort(key=lambda r: (r.get(col["Field"]) is None, r.get(col["Field"]) or ""), reverse=rev)
    total = len(rows)
    rows  = rows[offset: offset + limit if limit else None]
    return rows, total

def _match(row, f):
    if "Filters" in f:
        res = [_match(row, s) for s in f["Filters"]]
        return all(res) if f.get("LogicalOperator", "AND") == "AND" else any(res)
    rv, v, op = row.get(f.get("Field", "")), f.get("Value"), f.get("Operator", "=")
    if op == "=":           return rv == v
    if op == "!=":          return rv != v
    if op == "<":           return rv is not None and rv < v
    if op == "<=":          return rv is not None and rv <= v
    if op == ">":           return rv is not None and rv > v
    if op == ">=":          return rv is not None and rv >= v
    if op == "IS NULL":     return rv is None
    if op == "IS NOT NULL": return rv is not None
    if op == "IN":          return rv in v
    if op == "NOT IN":      return rv not in v
    if op == "LIKE":
        p = "^" + re.escape(v).replace(r"\%", ".*").replace(r"\_", ".") + "$"
        return bool(re.match(p, str(rv or ""), re.IGNORECASE))
    return False
```

## Error Codes

| Code | Condition |
|---|---|
| `XBASE_CONNECTION_INVALID` | Connection not open |
| `XBASE_SCHEMA_TABLE_NOT_FOUND` | Table does not exist in `_schema.json` |
| `XBASE_SCHEMA_COLUMN_MISSING` | A requested column does not exist in the table definition |

- `XBase-Database-Connect`
- `XBase-Query-Filter` — optional
