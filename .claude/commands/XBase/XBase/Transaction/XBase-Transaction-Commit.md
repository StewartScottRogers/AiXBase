# XBase-Transaction-Commit

Commit an open transaction by moving all workspace files over their live counterparts
and deleting the transaction workspace directory. Uses `File.Move` for atomic
single-volume replacement.

## Inputs

| Parameter | Type | Required | Default | Description |
|---|---|---|---|---|
| `TransactionName` | string | yes | — | Logical alias registered by `XBase-Transaction-Begin` |

## Outputs

```json
{
  "Success": true,
  "TransactionName": "<alias>",
  "CommittedAt": "<ISO-8601>"
}
```

## Steps

1. Look up `TransactionName` in the session; if not found, return `XBASE_TRANSACTION_NOT_OPEN`
2. Resolve the transaction workspace: `{DatabasePath}/_txn_{TransactionName}/`
3. Verify the workspace directory exists on disk; if not, return `XBASE_TRANSACTION_NOT_OPEN`
4. For each file in `_txn_{TransactionName}/` (excluding `sp_*/` savepoint subdirectories):
   - `File.Move(workspaceFile, liveFile, overwrite: true)` — atomic replacement on the same volume
5. Delete any `sp_*/` savepoint subdirectories inside the workspace: `Directory.Delete(sp_*/, recursive: true)`
6. `Directory.Delete(_txn_{TransactionName}/, recursive: true)`
7. Deregister `TransactionName` from the session
8. Return `CommittedAt`

## Script Templates

### PowerShell

```powershell
# Generated by XBase-Transaction-Commit
param($DatabasePath, $TransactionName)
$workspace = "$DatabasePath\_txn_$TransactionName"
Get-ChildItem -Path $workspace -File | ForEach-Object {
    Move-Item -Path $_.FullName -Destination "$DatabasePath\$($_.Name)" -Force
}
Remove-Item -Path $workspace -Recurse -Force
```

### Python

```python
# Generated by XBase-Transaction-Commit
import pathlib, shutil

def commit_transaction(db_path, transaction_name):
    dbpath    = pathlib.Path(db_path)
    workspace = dbpath / f"_txn_{transaction_name}"
    for f in workspace.iterdir():
        if f.is_file():
            f.replace(dbpath / f.name)
    shutil.rmtree(workspace)
```

## Error Codes

| Code | Condition |
|---|---|
| `XBASE_TRANSACTION_NOT_OPEN` | No active transaction workspace for that name |
