# XBase-Database-Drop

Delete a database directory and all its contents permanently.

## Inputs

| Parameter | Type | Required | Default | Description |
|---|---|---|---|---|
| `DatabaseName` | string | yes | — | Name of the database directory under `XBaseFiles/` |
| `ConfirmDrop` | bool | yes | — | Must be `true`; guards against accidental data loss |

## Outputs

```json
{
  "Success": true,
  "DatabasePath": "<absolute path>",
  "DroppedAt": "<ISO-8601>"
}
```

## Steps

1. If `ConfirmDrop` is not `true`, return `XBASE_DROP_NOT_CONFIRMED`
2. Resolve `XBaseFiles/{DatabaseName}/` to an absolute path; reject if the name contains `..`, `/`, `\`, or null bytes — return `XBASE_DATABASE_PATH_INVALID`
3. Verify the directory exists; if not, return `XBASE_DATABASE_NOT_FOUND`
4. Deregister any session connections whose `DatabasePath` matches this directory
5. `Directory.Delete(DatabasePath, recursive: true)` — removes all `.dbf`, `.ndx`, `_meta.json`, `_schema.json`, `_txn_*/`, and `backups/` subdirectories in one operation
6. Return `DatabasePath` and `DroppedAt`

## Script Templates

### PowerShell

```powershell
# Generated by XBase-Database-Drop
param($DatabasePath)
if (-not (Test-Path $DatabasePath -PathType Container)) {
    throw "XBASE_DATABASE_NOT_FOUND"
}
Remove-Item -Path $DatabasePath -Recurse -Force
```

### Python

```python
# Generated by XBase-Database-Drop
import pathlib, shutil

def drop_database(db_path):
    p = pathlib.Path(db_path)
    if not p.is_dir():
        raise FileNotFoundError("XBASE_DATABASE_NOT_FOUND")
    shutil.rmtree(p)
```

## Error Codes

| Code | Condition |
|---|---|
| `XBASE_DROP_NOT_CONFIRMED` | `ConfirmDrop` was not `true` |
| `XBASE_DATABASE_NOT_FOUND` | Directory does not exist at the resolved path |
| `XBASE_DATABASE_PATH_INVALID` | Name contains illegal characters or path traversal sequences |

## Dependencies

- `XBase-Database-Connect` — any open connections are deregistered automatically before the drop
