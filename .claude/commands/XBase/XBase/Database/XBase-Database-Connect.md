# XBase-Database-Connect

Validate an existing database directory and register a named connection alias for the session.

## Inputs

| Parameter | Type | Required | Default | Description |
|---|---|---|---|---|
| `DatabaseName` | string | yes | — | Name of the database directory under `XBaseFiles/` |
| `ConnectionName` | string | yes | — | Logical alias used by all subsequent skills to identify this connection |

## Outputs

```json
{
  "Success": true,
  "ConnectionName": "<alias>",
  "IsOpen": true
}
```

## Steps

1. Reference the `XBaseRuntimeContext` established by `XBase-Runtime-Detect`. If no runtime context is present in the session, run `XBase-Runtime-Detect` first.
2. Resolve `XBaseFiles/{DatabaseName}/` to an absolute path.
3. Verify the directory exists; if not, return `XBASE_DATABASE_NOT_FOUND`.
4. Read `_meta.json`; verify it is valid JSON and contains `XBaseVersion`; if not, return `XBASE_DATABASE_CORRUPT`.
5. Read `_schema.json`; verify it is valid JSON with `Tables` and `Indexes` arrays; if not, return `XBASE_DATABASE_CORRUPT`.
6. If `ConnectionName` is already registered in the session, return `XBASE_CONNECTION_NAME_IN_USE`.
7. Register `ConnectionName → DatabasePath` mapping in the current session context.
8. Return `ConnectionName` and `IsOpen: true`.

## Script Templates

### PowerShell

```powershell
# Generated by XBase-Database-Connect
$dbPath = Join-Path $PSScriptRoot "XBaseFiles" "<DatabaseName>"
if (-not (Test-Path $dbPath)) { throw "XBASE_DATABASE_NOT_FOUND" }
$meta   = Get-Content -Raw (Join-Path $dbPath "_meta.json")   | ConvertFrom-Json
$schema = Get-Content -Raw (Join-Path $dbPath "_schema.json") | ConvertFrom-Json
if (-not $meta.XBaseVersion)        { throw "XBASE_DATABASE_CORRUPT" }
if ($null -eq $schema.Tables)       { throw "XBASE_DATABASE_CORRUPT" }
# Register in session: $XBaseConnections["<ConnectionName>"] = $dbPath
$XBaseConnections["<ConnectionName>"] = $dbPath
Write-Output "{ `"Success`": true, `"ConnectionName`": `"<ConnectionName>`", `"IsOpen`": true }"
```

### bash

```bash
#!/usr/bin/env bash
# Generated by XBase-Database-Connect
DB_PATH="$(pwd)/XBaseFiles/<DatabaseName>"
[ -d "$DB_PATH" ] || { echo "XBASE_DATABASE_NOT_FOUND"; exit 1; }
python3 -c "import json,sys; d=json.load(open('$DB_PATH/_meta.json')); sys.exit(0 if 'XBaseVersion' in d else 1)" || { echo "XBASE_DATABASE_CORRUPT"; exit 1; }
# Session state: export XBASE_CONN_<ConnectionName>="$DB_PATH"
export XBASE_CONN_<ConnectionName>="$DB_PATH"
echo "{\"Success\":true,\"ConnectionName\":\"<ConnectionName>\",\"IsOpen\":true}"
```

### Python

```python
# Generated by XBase-Database-Connect
import os, json

db_path = os.path.join(os.getcwd(), "XBaseFiles", "<DatabaseName>")
if not os.path.isdir(db_path):
    raise FileNotFoundError("XBASE_DATABASE_NOT_FOUND")
with open(os.path.join(db_path, "_meta.json"), encoding="utf-8") as f:
    meta = json.load(f)
if "XBaseVersion" not in meta:
    raise ValueError("XBASE_DATABASE_CORRUPT")
with open(os.path.join(db_path, "_schema.json"), encoding="utf-8") as f:
    schema = json.load(f)
if "Tables" not in schema:
    raise ValueError("XBASE_DATABASE_CORRUPT")
# Session state stored in caller's context: connections["<ConnectionName>"] = db_path
print(json.dumps({"Success": True, "ConnectionName": "<ConnectionName>", "IsOpen": True}))
```

## Error Codes

| Code | Condition |
|---|---|
| `XBASE_DATABASE_NOT_FOUND` | The database directory does not exist at the resolved path |
| `XBASE_DATABASE_CORRUPT` | `_meta.json` or `_schema.json` is missing or contains invalid JSON |
| `XBASE_CONNECTION_NAME_IN_USE` | A connection with that alias is already registered |

## Dependencies

- `XBase-Runtime-Detect` — session must have a runtime context before this skill executes.
- `XBase-Database-Initialize` — must have been called first to create the database directory.
