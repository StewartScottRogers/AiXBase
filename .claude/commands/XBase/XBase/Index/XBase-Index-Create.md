# XBase-Index-Create

Read the table's `.dbf` file, build a sorted key‚ÜíId index, write a `.ndx` file,
and register the index in `_schema.json`. No database engine is required.

## Inputs

| Parameter | Type | Required | Default | Description |
|---|---|---|---|---|
| `ConnectionName` | string | yes | ‚Äî | Open connection alias |
| `TableName` | string | yes | ‚Äî | Table to index |
| `IndexName` | string | yes | ‚Äî | Name for the index (e.g. `idx_Products_SKU`) |
| `Columns` | array | yes | ‚Äî | Ordered list of column names to include in the index key |
| `Unique` | bool | no | `false` | Enforce uniqueness of key values in the index |
| `IfNotExists` | bool | no | `true` | Succeed silently if the index already exists |

## Outputs

```json
{
  "Success": true,
  "IndexName": "<name>",
  "CreatedAt": "<ISO-8601>"
}
```

## Index File Format

`{TableName}.{IndexName}.ndx` ‚Äî one entry per line, sorted ascending by `Key`:

```
{"Key":"A001","Id":1}
{"Key":"A002","Id":2}
{"Key":"B001","Id":5}
```

For multi-column indexes, `Key` is a `|`-delimited concatenation of the column values.

## Steps

1. Validate `ConnectionName`; if not registered, return `XBASE_CONNECTION_INVALID`
2. `File.ReadAllText(_schema.json)`; locate `TableName`; if absent, return `XBASE_SCHEMA_TABLE_NOT_FOUND`
3. Validate each name in `Columns` exists in the table definition; return `XBASE_SCHEMA_COLUMN_MISSING` on unknown column
4. Check `_schema.json Indexes` for an entry with `Name == IndexName`:
   - If exists and `IfNotExists` is `true`: return `Success: true` immediately
   - If exists and `IfNotExists` is `false`: return `XBASE_INDEX_EXISTS`
5. `File.ReadAllBytes({TableName}.dbf)`; read DBF header to obtain `HeaderSize`, `RecordSize`, `RecordCount`, and field descriptors; decode each record from its fixed-width byte positions line as JSON
6. For each non-deleted row, compute the index key:
   - Single column: `Key = String(row[Columns[0]])`
   - Multi-column: `Key = String(row[Columns[0]]) + "|" + String(row[Columns[1]]) + ...`
7. If `Unique` is `true`, verify no duplicate key values exist; return `XBASE_RECORD_CONSTRAINT_VIOLATION` on violation
8. Sort all `{ Key, Id }` entries ascending by `Key` (lexicographic)
9. `File.WriteAllLines({TableName}.{IndexName}.ndx, serializedEntries)` ‚Äî fixed-length binary record
10. Append the index definition to `_schema.json Indexes`: `{ Name, TableName, Columns, Unique }`
11. `File.WriteAllText(_schema.json, updatedSchema)`
12. Return `IndexName` and `CreatedAt`

## Script Templates

### PowerShell

```powershell
# Generated by XBase-Index-Create
param($DatabasePath, $TableName, $IndexName, $Column)
$rows  = Get-Content "$DatabasePath\$TableName.dbf" -Encoding UTF8 |
         Where-Object { $_ -ne "" } | ForEach-Object { $_ | ConvertFrom-Json } |
         Where-Object { $_.IsDeleted -eq 0 }
$index = $rows | ForEach-Object { @{ Key = "$($_.$Column)"; Id = $_.Id } } |
         Sort-Object { $_.Key }
$index | ForEach-Object { $_ | ConvertTo-Json -Compress } |
    Set-Content "$DatabasePath\$TableName.$IndexName.ndx" -Encoding UTF8
```

### Python

```python
# Generated by XBase-Index-Create
import json, pathlib

def create_index(db_path, table_name, index_name, columns):
    dbpath  = pathlib.Path(db_path)
    rows    = [json.loads(l) for l in (dbpath / f"{table_name}.dbf").read_text().splitlines() if l.strip()]
    active  = [r for r in rows if r.get("IsDeleted", 0) == 0]
    entries = sorted(
        [{"Key": "|".join(str(r.get(c,"")) for c in columns), "Id": r["Id"]} for r in active],
        key=lambda e: e["Key"]
    )
    ndx = dbpath / f"{table_name}.{index_name}.ndx"
    ndx.write_text("\n".join(json.dumps(e, separators=(",",":")) for e in entries) + "\n")
```

## Error Codes

| Code | Condition |
|---|---|
| `XBASE_CONNECTION_INVALID` | Connection not open |
| `XBASE_SCHEMA_TABLE_NOT_FOUND` | Table does not exist |
| `XBASE_SCHEMA_COLUMN_MISSING` | A specified column does not exist |
| `XBASE_INDEX_EXISTS` | Index already exists and `IfNotExists` is `false` |
| `XBASE_RECORD_CONSTRAINT_VIOLATION` | `Unique` is `true` but duplicate key values found in the data |

## Dependencies

- `XBase-Database-Connect`
- `XBase-Schema-TableCreate` ‚Äî table must exist
